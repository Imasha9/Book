# Backend -Node.js + Express.js
# Frontend-React.js

# Setup
## 1.Clone the Repository:
   git clone https://github.com/Imasha9/Book.git
   
### 2.Install Dependencies:
-For the backend:  

  cd backend  
  
  npm install  
  
-For the frontend:  

  cd frontend  
  
  npm install  
  
  
 ### 3.Start the backend server:
  cd backend  
  
  npm start  
  
 ### 4.In a separate terminal, start the frontend server:
  cd frontend  
  
  npm start
